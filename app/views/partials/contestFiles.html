Pictures

Creating a contest will causes two directories to be created.
  • Original: This is where a copy of the pictures being added will go. They will never be modified.
  • Testdata: probably miss-named, any changes be done to files in this directory.
You should next
  • Email Address: this is necessary when it comes time to "Email Contest"
  • Picture Age Limit: Any picture taken before this date should not be in the contest
  • Choose Picture: Click the Choose Picture - select pictures to add. They will be shown in a table.
  • From the table you can see some basic information:
    • Filename: The original name unless changed via the "R" button
    • Name: As parsed from the Filename
    • Title: As parsed from the Filename
    • Width: Invalid if > 1024
    • Height: Invalid if > 1024
    • Copyright: If invalid can click the Copyright button in which one will be generated
    • Size: Invalid if > 300 K
  • Table buttons: only available when directory is Testdata.
    • R: rename file
    • Copyright: generate a valid copyright
    • D: delete picture
Once you are ready
  • Generate Contest: This will create the necessary copies of the directories with zip file to be emails
  • Email Contest: This will send 4 emails: one for each zipped directory. Be very patient this will take 10 to 15 minutes.
